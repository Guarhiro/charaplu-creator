# キャラぷクリエイター 🎲

Gemini Gem「キャラぷ」用プロンプトを16面ダイスで完全自動生成するWebアプリ。

## 使い方
1. [サイトにアクセス](https://あなたのユーザー名.github.io/charaplu-creator/)
2. 🎲「ダイスを振る！」でプロンプト自動生成
3. ⚙️ Gemini APIキーを設定するとAI強化機能が使える
4. 📥 Markdownでダウンロード → キャラぷにコピペ

## AI強化機能（APIキー必要）
- ✨ プロンプト推敲
- ✨ キーワードブック全拡張
- ✨ 最初の一言強化
- ✨ キャラ深掘り
- ✨ ランキング最適化
- 🎨 立ち絵プロンプト生成

APIキーは [Google AI Studio](https://aistudio.google.com/apikey) で無料取得できます。
